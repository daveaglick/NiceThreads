Detailed Description

template<T>
class NiceThreads::ReadOnlySyncObject< T >

Mimics the semantics of a readonly field in that the underlying object cannot be changed once constructed. Uses a ReaderWriterLockSlim locking object if an alternate Locker isn't specified.

Template Parameters:
TThe type of the object to be wrapped.
